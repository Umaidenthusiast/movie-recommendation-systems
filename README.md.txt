#  Movie Recommendation System

A simple movie recommendation system built with **Python** and **Flask** that suggests movies based on similarity.  
The system uses **content-based filtering** and provides an interactive web interface.

---

##  Features
- Recommend movies similar to user input
- Flask-powered web interface
- Organized project structure
- Easy to extend with new models

---

##  Tech Stack
- Python (Pandas, NumPy, Scikit-learn)
- Flask (for web framework)
- HTML/CSS (frontend templates)
